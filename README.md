# Runbook Lens

Runbook Lens는 운영자가 장애나 고객 이슈를 확인할 때 로그를 하나씩 뒤지지 않아도 되게 해주는 로컬 로그 점검 앱입니다.

사용자는 Windows에서 로그 파일이나 폴더를 선택하고, 앱은 예외·타임아웃·인증 실패·데이터 손실·결제/주문 오류 같은 운영 위험 신호를 자동으로 찾아줍니다. 찾은 결과는 심각도, 규칙명, 파일 경로, 라인 번호, 미리보기와 함께 보여주며, 필요한 항목을 골라 메모를 붙인 뒤 Markdown 또는 JSON 보고서로 내보낼 수 있습니다.

핵심은 **로그 분석과 인수인계 보고서 작성을 빠르게 끝내는 것**입니다. 외부 서버로 로그를 업로드하지 않고 로컬에서 스캔하므로, 민감한 운영 로그를 다루는 작은 팀이나 개인 개발자가 장애 대응·QA·운영 점검 시간을 줄이는 데 쓸 수 있습니다.
